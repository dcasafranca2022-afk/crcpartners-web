## Publicación

1. Descomprimir el ZIP.
2. Copiar `index.html`, `README.md` y la carpeta `assets` dentro de:
   `Documentos/GitHub/crcpartners-web`
3. Reemplazar archivos.
4. En GitHub Desktop:
   Commit: `Nueva presentación V3.0`
5. Push origin.
6. Revisar con Ctrl + F5:
   https://dcasafranca2022-afk.github.io/crcpartners-web/
